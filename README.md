Задача
Реализовать локальный поиск для нахождения приближенного решения задачи о независимом множестве.

Требования:
Начать со случайного независимого множества
Пытаться добавлять вершины, если они не смежны с текущим множеством
Вывести найденное независимое множество и его размер
Входные данные:
Граф с 12 вершинами

Определение
Независимое множество в графе — это множество вершин, никакие две из которых не соединены ребром. Задача о максимальном независимом множестве состоит в нахождении независимого множества максимального размера. Эта задача является NP-трудной.

Алгоритм локального поиска
Объяснение работы алгоритма:

Инициализация: Создаем случайное независимое множество, последовательно добавляя вершины если они не смежны с уже добавленными

Улучшение решения: В основном цикле пытаемся добавить новые вершины в множество, проверяя для каждой:

Ключевая проверка: if neighbor in current_set: - если любой сосед уже в множестве, вершину добавить нельзя

Если все проверки пройдены - добавляем вершину и продолжаем поиск

Завершение: Алгоритм останавливается, когда нельзя добавить ни одну новую вершину

Временная сложность
Объяснение средней сложности:

Инициализация: 
Сумма по всем вершинам: O(|E|)
Количество итераций: в среднем O(|V|), так как на каждой итерации добавляем максимум 1 вершину
На каждой итерации:
перебираем все вершины: O(|V|)
для каждой вершины проверяем всех соседей: O(deg(v))
Суммарно за итерацию: O(|V| + |E|)
Итого средняя сложность:
Средняя: O(|V| × |E|)

|V| - количество вершин

|E| - количество ребер
